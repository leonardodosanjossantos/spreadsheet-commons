# Spreadsheet Commons
Project with rules in common between api and processor projects;

## How it Works 
Project generate a jar in order to improve api and processor projects and avoid coupling and duplicated code.

## Configuration
* Java 8 or above
* Maven
* RabbitMq for messages
* Mysql Client

## How to generate
```bash
mvn clean install
```

## Main Frameworks

* Spring Boot
* Lombook

## Test Frameworks 

* Junit
* Mockito
* Matcher
